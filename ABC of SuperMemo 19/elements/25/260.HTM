Title: 键盘快捷键

鼠标快捷键
●左键单击: 编辑 组件 (例如，编辑文本内容)
●Shift+左键单击: 编辑文件 (例如，编辑图片)
●Ctrl+左键单击: 查看和当前 组件 相关的文件(例如，在默认的图片阅览器中阅览图片)。在 HTML 组件中, Ctrl+左键单击和在 Internet Explorer 有一样的效果 (例如，选择图片、文本，或者打开超文本链接什么的)。
●Alt+左键单击: 切换 组件 (or 元素)的 展示(presentation), 编辑(editing) and 拉取(dragging) 模式。


单一键

A: 编辑第一个答案
E: 编辑文本
Q: 编辑第一个问题
Del: 删除当前展示的 元素 (当您处于 编辑模式的时候，您应该要使用 Shift+alt+del 来删除当前元素)
Ins
○在 内容(contents) 窗口中: 插入元素
○在 浏览器中: 选择或者不选择 元素
○在 计划: 在当前选择的列中插入新的活动
Tab
○在对话窗口: 移动到下一个按钮，下一个检验栏中，等等.
○在 元素窗口: 切换到下一个组件
○在任务列表(tasklist)管理器(manager): 选择新的 任务列表
Enter
○在 浏览器: 在元素窗口中展示当前 元素







Function 键

F1: 帮助
F2: Tools : Memory : 记忆状况
F3：搜索查找
F4: 在 任务列表管理器 打开当前正在用的 任务列表
F5: 应用 Warrior 布局而不需要更改默认布局
F6：过滤网页

在 HTML/RTF 组件中, 过滤网页资源，删除在增量阅读中的某些格式（HTML样式） 去除导入HTML内不适用于SuperMemo的格式 (例如，移除掉使 渐进阅读 变得困难的格式)
在 文件中 : 打开收集(collection) 对话箱，重命名当前的 收集
F7: 工具(Tools) : 记忆(Memory) : 4D 图
F8: 并不会一直使用。这个按键将会在各种测试、发行版中使用。
F9: 查看和 组件 相关的文件。例如，如果您使用 MS Office ，您如果在富文本组件上按下 F9 ，那么您将会以 MS Word 来编辑它，而如果您按下在 HTML 组件上按下 F9 的话，那么您将可以以默认浏览器查看当前 HTML 文件。 (您还可以: Ctrl+F9 来编辑文件)
F11：随机复习

F12: 睡眠和学习的时间轴 (工具 : 睡眠图表)
